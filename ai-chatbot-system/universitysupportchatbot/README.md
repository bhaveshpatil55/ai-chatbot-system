# University Information Bot

A Python-based university support chatbot that helps students with common university-related queries using a rule-based approach.

## Features
- Admission information
- Scholarship details
- Hostel facility information
- Library timings
- Examination schedule
- Result information
- Placement support
- Course information
- Handles small spelling mistakes using difflib

## Technologies Used
- Python
- difflib (built-in Python library)

## How to Run

1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the program:

```bash
python university_support_chatbot.py
```

## Example Queries
- admission
- scholarship
- hostel
- library
- exam
- result
- placement
- fees
- course
