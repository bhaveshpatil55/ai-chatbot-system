# Student Support Bot

A rule-based student support chatbot that assists students with common college-related queries and information.

## Features
- Exam information support
- Assignment submission guidance
- Library timing information
- Attendance details
- Result information
- Handles small spelling mistakes using difflib

## Technologies Used
- Python
- difflib (built-in Python library)

## How to Run

1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the program:

```bash
python student_support_chatbot.py
```
